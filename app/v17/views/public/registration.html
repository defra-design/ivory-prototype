What you need to do

Seller information

You can sell this item based on the details in this registration document being correct.

This registration expires when you sell the item.

The registration will not expire if you hire out the item.

You can give the registration number to a buyer so they can check the item before they buy it.

Report changes

You must tell Animal and Plant Health Agency (APHA) if:

  • any information about you or the item changes
  • the item no longer meets the criteria for registration


Buyer information

Before you buy an ivory item, you must check:

  • the item is registered - and matches the description in the registration
    • {% if data['exemptionChoice'] == 'type1' %}
  • you agree the item was made before 1947 and has less than 10% ivory
    • {% endif %} {% if data['exemptionChoice'] == 'type2' %}
  • you agree the item is a musical instrument made before 1975 and has less than 20% ivory
    • {% endif %} {% if data['exemptionChoice'] == 'type3' %}
  • you agree the item is a portrait miniature made before 1918 and has a surface area less than 320 square centimetres
    • {% endif %}

Use check before you buy an ivory item to view details of the item.
www.gov.uk/guidance/check-before-you-buy-ivory

After the sale

As the new owner you must:

  • contact APHA to give them your name and address
  • register the item again if you plan to sell it

Read the guidance

Find out what to do if you want to sell, hire or buy ivory items https://www.gov.uk/guidance/a-guide-to-selling-hiring-and-buying-items-containing-ivory
